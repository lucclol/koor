# 🍺 The Banquet Calculator

> *You earned it. Find out how many.*

A simple, beautifully designed calculator that tells you how many **Coors Banquet** beers you've earned based on calories burned in a day.

**Live demo:** [your-vercel-url.vercel.app](https://your-vercel-url.vercel.app)

---

## Features

- Enter calories burned (cal or kcal)
- Activity type selector
- Calculates beers earned at **149 cal per Coors Banquet**
- Shows leftover calories, beer visual, and a color-coded verdict
- Snarky verdict messages
- Fully responsive

## Tech Stack

Pure HTML, CSS, and vanilla JavaScript — no dependencies, no build step.

---

## Deploy to Vercel

### Option 1 — One-click via Vercel CLI

```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub + Vercel Dashboard

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your GitHub repo
4. Vercel auto-detects the static site — click **Deploy**

That's it. No build configuration needed.

---

## Local Development

No build step required. Just open `index.html` in your browser:

```bash
open index.html
# or
npx serve .
```
